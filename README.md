# Browser Bug Searcher

Demo: https://browser-issue-tracker-search.appspot.com

## Contributing

First, install the [App Engine Python SDK](https://cloud.google.com/appengine/downloads). Then run:

    npm install

To deploy, run:

    npm run deploy
